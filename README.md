## 本地启动
请确保本地有nodejs环境

    yarn install
    yarn docs:dev
    
访问 http://localhost:8080

## 修改路由
请打开/doc/.vuepress/config.js
编辑sidebar部分内容

## 静态文件
请放置在/doc/.vuepress/public
/public 目录下内容引用请使用 /xxx
或使用相对路径，放置于md文件相同目录./
